Usage Instructions:

  1. Start Animation: Click the "Start Animation" button to begin the noise text effect
  2. Change Text: Use the "Change Text" button to cycle through different text examples
  3. Stop Animation: Click "Stop Animation" to pause the effect
  4. Responsive: Resize your browser window to see automatic adaptation

Code Example:

import AnimatedNoiseText from 'animated-noise-text';

const canvas = document.getElementById('canvas');
const animation = new AnimatedNoiseText(canvas, {
  text: 'HELLO WORLD',
  cellSize: 2,
  stepMs: 32
});

animation.start();
